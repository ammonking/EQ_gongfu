    掌控自己：个人管理系统

    Ammon 2020-06-20 07:45:51 个人管理

    这本书无疑是自我管理必读的一本书,如果让我只能选三本枕边书,那这一本是必选的. 结合<自我管理提升训练营>可以让自己在自我管理和任务管理方面取得质的飞跃!

    # 内容提要

    本书以三个人的故事为主线，讲述了个人管理系统搭建的全过程，其中包括个人管理的若干方法，概括为两个系统模型，分别是“WATCH”（表盘模型）和“IPO”（表带模型）。“WATCH”（表盘模型）是由WISH（愿望目标）、ACTION（行动计划）、TIME（时间记录）、COMMENTS（总结反思）、HYPER ENERGY（充沛精力）等五个元素构成的，这个表盘通过强大的行动力让整个系统顺畅运转起来。仅有“WATCH”（表盘模型）还不够，我们还需要“IPO”（表带模型）来不断精进个人管理系统的方法。每个人的需求和场景都在不断发生变化，需要通过INPUT（输入）和OUTPUT（输出）来带动整个表盘的转动，使得个人管理体系越来越精进和适用。有了这两大模型，可以让我们的个人管理体系有螺旋式的上升。

    “WATCH”（表盘模型）代表的是5个个人管理系统的元件，分别是WISH（愿望目标）、ACTION（行动计划）、TIME（时间记录）、COMMENTS（总结反思）和HYPER ENERGY（充沛精力）。这个表盘通过强大的行动力让整个系统顺畅运转起来。这本书的前半部分花费大量的篇幅来讲述如何进行时间开销记录，如何制订自己的日计划和周计划，如何制订自己的年度目标并分解年度目标到月度计划，如何书写日志并进行反思，如何进行精力管理。这些方法都是经过自己实践后的经验总结，可能并不适用于每个人，但是至少可以带给你一些启发。

    有了“WATCH”（表盘模型）还不够，我们需要“IPO”（表带模型）来不断精进个人管理系统的方法。每个人的需求和场景都在不断发生变化。需要通过INPUT（输入）和OUTPUT（输出）来带动整个表盘的转动，使得个人管理体系越来越精进和适用。拆书法是我找到的特别适合自己的一种方法，可以让我在知识、信息、经验中找到可以利用的内容输入到大脑中，通过拆书的方法学以致用，应用到自己的行动中去。

    有了这两大模型，可以让我们的个人管理体系有螺旋式的上升。

    # WATCH和IOP的含义

    我们先来分享一下WATCH表盘中四个维度的含义。

    WATCH表盘12点钟方向是Wish，代表的是愿景目标。我们进行个人管理时，必须要有一个清晰的画面告诉我们将要去向何方，成为一个什么样子的人，完成什么样的愿望，达成什么样的目标。它是我们所有行动的方向指引，告诉我们需要按照自己的意愿过这一生。”

    “WATCH表盘3点钟方向是Action，代表的是任务计划。当我们确定了自己的方向和想要成为的人，我们需要依据这个愿景目标制订出具体的任务计划。它比愿景目标更加具体、细致。是在去向愿景目标的道路上阶段性的里程碑。我们说愿景目标是让我们选择正确的路走，那任务计划就是让我们在这条正确的路上小步快跑。”

    “WATCH表盘6点钟方向是Time record，代表的是时间记录。我们所有的计划任务都有一个基本的要素‘时间开销’。它是个人管理中最底层的一项基础工作。当我们用数据化的方式呈现出来每时每刻的样子，可以更加深刻地体会到执行力的状态。”

    “WATCH表盘9点钟方向是Comments，代表的是总结反思。经过时间开销记录的数据分析后，我们可以通过总结的方式，反思在当下任务执行的情况如何。我们需要如何进行改善，愿望目标是否需要调整。”

    “WATCH表盘中间的位置是Hyper energy，代表的是充沛精力。这是WATCH模型的核心，也是个人管理体系的核心。只有拥有充沛的精力，才能将表盘正常运转起来而不至于停止。它是为所有个人管理四个分支‘愿景目标、行动计划、时间开销、总结反思’提供源源不断的动力。”

    再来看看IOP表带模型

    IPO表带模型包括如下三个部分。

    I：Input，代表输入。

    P：Power of action，代表执行力。

    O：Output，代表输出。

    “IPO模型左边的表带是I，代表着输入Input。在构建自己的个人管理体系时需要不断地吸收知识，获得前辈们的方法。输入的方式包括阅读、学习、听课、实践等。

    IPO模型右边的表带是O，代表着输出Output。在构建自己的个人管理体系时需要不断地分享知识，学以致用，以教为学是最好的学习方式。输出的方式包括写作、教学、上课、分享等。”

    “IPO模型中间是表盘的指针Point，代表着执行力Power of action，由在中心的H充沛的精力提供源源不断的动力。它不仅仅是提供表盘四个维度的执行力，也提供IPO从输入到输出的执行力。”

    “输入和输出相结合，会让我们的表盘更加稳固在我们的手臂上。输出会带动输入，输入同样可以带动输出。在构建个人管理系统时，我们需要不断强化输入和输出能力。只有这样才能将个人管理体系搭建得更加完善，并且适合自己。”

    # WATCH+IPO模型构成完成的个人管理体系

    “WATCH表盘模型的四个要素是从纵向的方式上来阐述个人管理体系的构架，从上到下分别是‘愿景目标，计划任务，时间开销，以及总结反思’。它是一个循环的系统，是由WATCH表盘中心的‘充沛精力’提供能量来源。与此同时，WATCH四个要素在某种程度上可以反作用于我们的精力，让我们可以持续下去的原动力往往来自于我们自身对于个人管理体系的不断深入。”

    “IPO表带模型的左右表带是从横向的方式上来阐述个人管理系统的构架，一边是输入，一边是输出，中间是高效的执行力。在这个横向的结构中，我们需要注意的是快速反馈。这里要注意两点，一是整个流程的完整度。先输入，再快速执行，最后是输出。不能缺少任何一个环节。二是整个流程的快速循环，从输入到输出的时间间隔尽量缩短，获得快速反馈，不断完善，螺旋式上升。”

    “介绍完毕两个模型后，大家对于个人管理体系有了初步的了解。下面我会具体来分享每个环节自己的经验和体会. 书中是按照由易到难的顺序进行的,所以在第一次阅读时会觉得懵, 在熟练掌握后作者的说法是可以从任意一点开始自己高效的个人管理.

    # 普世的智慧

    这里介绍的个人管理方法是查理·芒格口中所说的“普世的智慧”，是一门特别基础的学科分类。如果要按照学科分类的方式，它一定是与语文、数学并列的基础学科。正是这个“普世的智慧”学科帮助高一帆搭建了最为底层的基础框架，在这个框架上每个人都可以任意填充自己想要做的事情，比如学习、阅读、写作、健身、社交、工作、居家等各领域的大小事情，都可以放在这个框架中。高效运行这个体系，可以让我们进行多任务进程的推进，同时平衡好工作和生活的黄金分割点。

    1.自律即自由。越自由的工作状态，越需要超强的自控力。

    2.焦虑是发现问题的钥匙。如果焦虑，看看自己出现了什么问题，如何解决。

    3.个人管理能力是基础技能。如果不知道学习什么，那就学习个人管理知识。

    # 重视自己的承诺

    在工作中,我们常常有这样的苦恼 --

    • 凡事有求必应，也不太懂得拒绝。慢慢地形成了随口答应的习惯，口头禅是“好，好的。”由于答应的事情越来越多，王成功渐渐发现自己很容易遗忘这些答应的事情。
    • 领导总是给自己安排很多临时的事情去处理。开始计划好的任务，都会被这些临时突发事件干扰。渐渐地，每次接到领导安排的临时任务，你都会火冒三丈，有时候也会跟领导发脾气。

    其实,这都是因为没做好承诺管理.

    同事给你的任务随口答应，答应后没有放在心上，这是没有管理好自己随口的‘承诺’;

    我们 有自己的规划,对于计划内的事情都有自己的规划。但是一碰到临时突发事情，就会慌乱、容易上火、发脾气。其实这些临时突发事情也是自己需要承担的‘承诺’。不要以为凡事都只能在计划范围内执行。我们每天都会碰到很多临时突发事情。

    想要管理好自己的承诺，有一个方法可以尝试，包括以下三个核心步骤。

    第一，清空大脑。

    第二，明确任务。

    第三，随时查看。

    首先，清空大脑。将自己大脑中的所有思绪从大脑中拿出来，放在一个客观、可靠的系统中。其次，明确任务。你必须知道你的任务是什么，需要采取什么行动来推动任务的进程。最后，随时查看。一旦决定了需要采取的行动，必须将这一系列的行动放在经常查阅的体系中随时查看。

    # 总结

    # 如何管理好自己的行动？

    • 横向管理：收集大脑中所有“未尽事项

    • 纵向管理：关注任务的流程化进度

    问自己：“这个任务（项目）我下一步应该做什么？”

    大多数人在当下都只能将自己的注意力放在一件事情上。如果我们在处理当前这件事情的时候，时不时地还想着其他的事情，就会降低自己的效率。如果我们能够很好地横向管理和纵向管理我们的行动集合，我们就不会被自己庞大的潜意识牵着鼻子走。我们会更加聚焦在当下的事情，集中精力完成它们。当我们完成当下的任务后，再拿出我们的可靠信任的系统，看看还有什么样的行动需要完成。就可以达到无压力的工作和生活状态了。”

    这就解释了之前承诺管理出现的哪两个问题.

    被领导安排去做一件事情，一会儿又被同事安排做另一件事情。东扯西拉，自己经常容易陷入同一时间内思考两件事情的情况。结果可想而知，两件事都没有做好。”

    我之前老爱发脾气。在有临时突发事件干扰我的时候，我是不愿意在我的大脑中同时放入两件事的，这样我会崩溃。如果我有一个可靠的系统将这些事情放进去。我的烦躁情绪应该可以得到很好的缓解。”

    # 个人管理方法的五个步骤

    个人管理方法的五个步骤，分别是收集、处理、组织、检视和执行 第一步：收集，一切引起我们注意的想法、灵感、信息与任务。 第二步：处理，对第一步收集到的这些信息进行处理，得出下一步行动集合。 第三步：组织，对这些具体的下ー步行动进行有效的组织整理。 第四步：检视，通过检视让个人管理体系运转起来。 第五步：执行，执行并完成下一步行动，最终完成任务。 依次往返重复

    个人管理方法的五个步骤只要在任意一个环节出现了短板，就会造成最后行动力的不足。我们要提高行动力，就必须提高这五个步骤对应的技能。针对这五个技能的训练，如果每次将注意力集中在一个技能的训练上，我们会完成得特别出色。比如集中精力完成‘收集’的这一环节，或安排两个小时全力以赴对所有收集的事务和信息进行‘处理’。这种训练的方式叫作‘刻意练习’，集中精力完成一个简单动作，会比集中精力完成一系列动作来得要好。”

    # 小结

    个人管理体系是职场人士的基础技能，包括五个步骤：收集、处理、组织、检视与执行。

    收集 1.收集工具有很多种，大体分为纸质版和电子版两大类。 2应将一切未尽事宜从大脑中赶到收集工具中。 3收集工具尽可能在10秒钟内能够拿出来进行记录。 4.收集工具数量不宜过多，最好不要超过两个。 5.收集工具应该定期进行清空。

    处理 流程第一步：在未来一段时间内要开始启动吗？ 流程第二步：这件事情复杂吗，需要分解吗 流程第三步：这件事情由谁来做最合理呢？ 流程第四步：这件事情是否马上就做，或者有固定执行日期吗?

    处理后的五张清单 1.项目清单：所有项目的集合清单。 2.日程清单：有特定日程安排的集合清单。 3.等待清单：等待他人执行任务的集合清单 4.下一步行动清单：所有简单行动的集合清单。 5.将来也许清单：不在本周内开始执行的集合清单。

    组织 按照情境的标签来分类管理 按地点不同划分为：家里、外出、办公室。 按工具不同划分为：电脑、手机。 按照优先级的标签分类：高优先级、中优先级、低优先级

    检视 1.日程表和下一步清单，每天需要检视多次 2.等待清单和项目清单，每天检视一次。 3.收集箱，每天早晨或者晚上检视一次。 4.将来也许清单，每周末检视一次

    执行 情境：当下所处什么情境。 时长：从现在开始有多少时 精力：当下的精力状況如何

    优先：从重要的事情开始。

    # 1.5 如何根据月度计划制定周计划?

    “其实很简单，月度计划的表格与周计划的表格一样。因此制订的过程也是相似的。可以梳理为下面4个步骤。”

    # 步骤1:分解月度计划为周计划

    首先根据月度计划表格的事项来分解对应的周计划事项。这里强调一个原则，先紧后松。前面两周的计划可以稍微安排得多一些。一是给自己留出突发事件的空间，二是给自己一些压力，先把重要的事情完成，再安排一些轻松的事情来做。

    # 步骤2：增加“将来也许”清单的临时事件

    然后将我们的Doit.im软件打开，看看在“将来也许”清单中有哪些事情可以安排在这一周来做。这些“将来也许”清单的事项都是我们平时收集而来的灵感或者遇到的突发事件。在制订周计划时，这个部分就是周计划内容的组成部分之一。

    # 步骤3：利用可利用小时数预估周计划数量

    确定完毕周计划后，要开始确定周计划的数量是否合适了。利用我们的可利用小时数来衡量周计划的数量。周计划由于范围更小，时间更短，比月计划的时间估算更加准确。高一帆说：“给大家举个我自己的例子。我一周的可利用小时数为70小时，乘以80%为56小时。我安排周计划的总时间长度就不能超过56小时。经过量化后，我们可以得到一个较为合理安排的周计划数量。”

    # 步骤4：将周计划任务分解至Doit.im软件中

    制订完毕周计划任务表格后，我们需要再将周计划的任务放置在Doit.im软件中。这个时候同样按照处理环节的思路进行分解，以项目化思维的方式将所有分解后的“下一步行动”放入Doit.im软件中。高一帆说：“我们经过了两次分解，第一次分解是从月度计划分解为周度计划事项。是从大项目分解为小项目的过程。第二次分解是从周计划任务分解为一个个项目为单位的下一步行动集合，从而与Doit.im软件结合起来。”

    # QA

    Q:是不是周计划需要每天都打开来看看呢？就跟月度计划每周都要打开来做更新的节奏一致。”

    A:“是的，每天我们需要查看周计划各个项目完成的进度情况。对于周计划有较大调整时，需要对周计划进行修改。不过这里建议大家只增加计划事项，不要删除计划事项。这样可以在最后对本周计划进行更好的复盘。再查看完毕周计划执行情况后，对第二天的计划进行制订。”

    # 1.6 自上而下的目标制定法则

    “上述的目标制订法则是自上而下的制订方法。我们可以从时间上来梳理一下。”

    1.年末，梳理价值观。找一个安静的环境，安排半天不被打扰的时间来梳理自己的价值观。这个环节每年安排一次就可以了，如果想要提高频率，在年中的时候也可以做一次。

    2.年末，魔法棒式地自由书写。找一个安静的环境，安排半天不被打扰的时间来自由书写100件事。这个环节每年安排一次就可以了，如果想要提高频率，在年中的时候也可以做一次。

    3.年末，绘制晋级路线图。从自由书写的100件事中选择出自己最想要做的10件事，并绘制相应的晋级路线图。让自己对于每个目标有一个大致的方向和思路。

    4.年末，制订年度目标。对自己未来一年角色的优先级排序，对自己各个晋级路线图进行筛选。确认年度目标最重要的三件事。并通过澄清六问让自己的目标更加清晰和立体。自己的情境发生巨大变化时，可以重新执行这个环节。确认下一阶段的三个大目标。

    5.月末，制订次月计划。根据年度目标的分解、自己要想要做的事情，以及其他临时突发的事情来制订次月的月度计划。

    6.周末，检视月度计划，制订次周计划。在每周末的时候检视月度计划表格，查看月度计划的进度。并对月度计划表进行调整。根据月度计划的执行情况，制订下周计划，并在Doit.im软件中制订一周的计划安排。

    使用的工具都很简单，就是大白纸、电子表格和Doit.im软件。只需要这三件武器就可以构建出我们的目标计划管理体系啦。”
