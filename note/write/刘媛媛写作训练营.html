# 刘媛媛写作训练营

# 为什么要学习写作？

  • 写作能力非常宝贵

    它是能跟你一辈子的金饭碗，是对自己最好的投资

  • 写作能带来可观收入

    稿件收入，广告收入，出书，授课，咨询，每年带来十万甚至百万的收入

  • 写作是一种机遇

    零成本转行，增加收入，增加个人影响力，更好的沟通

# 改变思维篇 -- 写作思维的三个重点

# 1 自媒体写作重点是“媒体”，而不是自说自话

  • 打破私密状态：不再是单单满足自己表达欲，不再让文字变成单纯的倾诉
  • 价值理论：把你的才华进行出售，形成对他人有价值的产品

# 2 以终为始

  1. 只有明确了目标，才能确定要做什么，怎么去做

    我们在写作前先要清楚自己的写作目标，一般的写作目标是这些：

    • 你希望别人阅读它
    • 你希望改变比人，让别人获得好的影响
    • 你希望别人帮你分享
    • 你希望他人掏钱

  2. 在清楚目标的基础上，找到你的写作目的和对方需求的交集

# 3 心里有他人

看破用户需求

每个读者都有其自身的需求，但是有些需求是通用的，从某些方面来说，写作，演讲，沟通也讲究心理学

人最重要的需求：获取幸福感， 而幸福感的来源有三个路径：

  • 胜任力

    人必须要感觉到自己有能力，感觉到自己强大，他才会有幸福感

  • 归属感

    人们都渴望被人爱，被人认可，这样才能感觉幸福

  • 自由感

    人们都希望所有的选择都是自己选的，不喜欢被要求或强迫

识别用户的情绪

洞悉用户的心理

很多自媒体大牛并没有高学历，但比高学历的人更优秀，原因就在于他们懂得用户需求！

# what：什么是自媒体好文章？

能够组我们目的的文章：被接受，被喜欢，被传播

# 自媒体人的必备思维

  1. 以终为始

  2. 生产者思维

    作为内容生产者，一定要有识别出好文章的能力

    什么是自媒体好文章，标准是什么？

    • 基础文章的标准

      1，表达精确，不复杂

      2，要有逻辑

      3，易于理解

      4，要有真诚和善意

    • 爆款文章的标准

      1 照顾读者情绪，说出读者想听的话，让读者感受到愉悦

      2 自带传播性，说出读者让对别人说的话，满足读者的社交需求

      3 提供信息整理能力，说出别人不知道，但想去了解的内容

      4 开启他人认知，说出对别人有用的话

# 刚开始写作要注意什么？

  1. 一定要忍住对自己的不满意，吧第一稿写出来
  2. 好文章不是一遍成型的，好文章是改出来的
  3. 写的不好，所以才要写，写得多了才能写好

# 快速开始的2个写作心态

  1. 先写出最容易的部分，把不知道怎么写的部分跳过去，都写完了，再公关难点
  2. 有价值就分享：你不是非要写出顶尖的东西，知道什么就写什么，分享给哪些比自己知道少一点的那个人就好

标题篇 -- 如何起个好标题

  1. 让用户感觉到这篇文章对他有用

    • 给用户价值感

    • 起到警示作用，警示一个危机的来临或是失去的风险

      《毕业生家长同屋，是我耽误了孩子》

  2. 让用户感觉到好奇

    • 答疑类标题

      《我月入3W，1万用来交智商税》

    • 对比类标题

      《李丹越火，我越想念王自健》

    • 反常类标题

      《崔永元骂人无数，但却单单发微博赞了他:他是个好人》

    • 故事类

      《你能不能慢点离开我？》

    • 新知类

      《比PUA更伤人的是救世主型伴侣》

    • 热点类

      《张韶涵崩溃大哭，人生没有容易二字》

    • 留白法

      《25岁时候你一定会被问。。。。》

  3. 让用户产生认同感

    1. 金聚类标题 --《我把你当老公，你把我当子宫》
    2. 对话类标题 -- 《我在北京有三套房，你有什么？滚》

  4. 让用户和你互动

    1. 用你开头，有话题感 -- 《给30岁的你100条小建议》
    2. 贴标签的标题，让读者对号入座 -- 《轮幽默，程序员从来没输过》
